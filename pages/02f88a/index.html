# tuples/list

Python 中的元组(Tuples)和列表(Lists)都是用于存储一组项目的数据结构, 但它们在以下方面有一些关键的区别:

  • 可变性(Mutability):

    • Lists 是可变的(Mutable), 这意味着你可以随时添加、删除或修改列表中的元素。使用方括号 [] 来表示列表。
    • Tuples 是不可变的(Immutable), 一旦创建了一个元组, 就不能更改其内容, 包括添加、删除或修改元素。使用圆括号 () 来表示元组。
  • 创建:
    • 列表通过方括号来创建, 如: my_list = [1, 2, 3]。
    • 元组通过圆括号来创建, 如: my_tuple = (1, 2, 3)。
  • 访问元素:
    • 你可以使用索引来访问列表和元组中的元素, 例如 my_list[0] 或 my_tuple[1]。
  • 迭代:
    • 你可以使用 for 循环来迭代列表和元组中的元素。
  • 方法和操作:
    • 列表提供了更多的方法和操作, 因为它们是可变的。你可以使用方法如 append(), extend(), remove(), pop() 等来修改列表。
    • 元组由于是不可变的, 提供的方法较少。你可以使用方法如 count() 和 index() 来检查元组中的元素。
  • 性能:
    • 由于元组是不可变的, 它们的性能通常比列表更高, 特别是在迭代和访问元素方面。
# 创建一个列表
my_list = [1, 2, 3]

# 创建一个元组
my_tuple = (1, 2, 3)

# 修改列表
my_list[0] = 10  # 合法, 列表是可变的
# 修改元组(将会引发TypeError异常)
# my_tuple[0] = 10  # 不合法, 元组是不可变的

# dict

  1. 使用键索引赋值:
  • my_dict["a"] = 42 更新键'a'对应的值
  1. 使用 update 方法:
  • dict1.update(dict2), update 方法可以用来将一个字典合并到另一个字典中
  1. 使用 dict 构造函数:
  • my_dict = dict(a=42, b=2)
  1. 使用字典解析: 字典解析允许你通过遍历其他可迭代对象的方式来更新字典。
  • my_dict = {k: v * 2 for k, v in my_dict.items()}, 将所有值乘以 2
  1. 使用 setdefault 方法: setdefault 方法可以用于在字典中添加新键-值对, 如果键已存在, 则不进行更新。
  • my_dict.setdefault("c", 42) 添加新键'c', 如果已存在则不修改
  1. 使用 dict 的 fromkeys 方法: fromkeys 方法可以用于创建一个具有指定键的新字典
keys = ["a", "b", "c"]
default_value = 0
my_dict = dict.fromkeys(keys, default_value)  # 创建一个新字典

# yield

yield 是 Python 中一个关键的关键字, 通常用于创建生成器(Generator)。生成器是一种特殊的迭代器, 允许你按需产生值, 而不是一次性生成所有值。这在处理大量数据或需要节省内存的情况下非常有用。

以下是关于 `yield` 关键字的一些重要概念:

  1. 生成器函数: 生成器函数是包含 yield 语句的函数。当你调用生成器函数时, 它不会执行函数体, 而是返回一个生成器对象。

  2. 生成器对象: 生成器对象可以像迭代器一样遍历, 但是它是惰性生成的。每次调用生成器的 __next__() 方法或使用 for 循环时, 生成器函数将从上次的 yield 语句处继续执行, 直到遇到下一个 yield 语句。

  3. yield 语句: yield 用于产生一个值, 并将控制权返回给调用者。生成器函数的状态将被冻结, 等待下一次迭代时继续执行。这使得你可以逐步生成数据, 而不必一次性生成所有数据。

以下是一个示例, 演示如何使用 yield 创建一个简单的生成器:

def simple_generator():
    yield 1
    yield 2
    yield 3

# 创建生成器对象
gen = simple_generator()

# 遍历生成器
for value in gen:
    print(value)

# reduce

functools.reduce 是 Python 标准库中的一个函数, 用于对可迭代对象中的元素进行累积计算。它的工作方式类似于累积函数, 可以用于执行类似于求和、乘积、最大值、最小值等操作。functools.reduce 接受两个参数:

  • 一个二元操作函数, 它接受两个参数并返回一个结果。这个函数用于定义要在累积过程中执行的操作。
  • 一个可迭代的对象, 其中包含要进行累积操作的元素。

functools.reduce 会从可迭代对象中按顺序取出元素, 将它们依次传递给操作函数, 并不断更新一个累积的结果值。最终, functools.reduce 返回累积的结果。

下面是一个示例, 演示如何使用 functools.reduce 来计算列表中元素的累积和:

import functools

# 定义一个二元操作函数, 用于计算两个数的和
def add(x, y):
    return x + y

# 要进行累积操作的列表
numbers = [1, 2, 3, 4, 5]

# 使用 functools.reduce 计算累积和
result = functools.reduce(add, numbers)

print(result)  # 输出 15, 即 1 + 2 + 3 + 4 + 5 的和
