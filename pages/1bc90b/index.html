# 资源

下载一个百度网盘，登录自己的百度网盘 然后添加用户名：英雄不问出路人 给师傅发送：ns 游戏然后会自动分享给你

# 游戏下载

下载游戏网址：https://www.bibgame.com/sgame/ (opens new window)

游戏年轮登陆账号: 用户名:游戏年轮 密码：liming123...切记请不要去修改密码，如查询到有修改密码者，不提供使用会员账号，且我们也会把之前的密码重新注销，会导致使用不了，谢谢

# 安装

把下载好的游戏软件放在内存卡的游戏文件夹 GAMES 里面，然后进入破解系统，选择相册进入，找到小 女孩的头像进入，选择内存卡安装，找到游戏文件夹，然后按照提示安装游戏即可。支持 nsp 和 xci 两种格式的游戏文件。安装游戏 时内存卡里面剩余的内存需要比游戏本体大。

# 安装游戏

这里应该是已经破解过的游戏下载，待验证

  1. 关机拔出内存卡，读卡器插上电脑。
  2. 下载 xci 或者 nsp 的游戏文件（不要下文件夹）到内存卡 agames 文件夹。
  3. 拔出内存卡，插回游戏机。
  4. 破解系统相册进入 atmoxl 进行安装，安装完安装包删除。

【游戏文件名字不能是中文，补丁及 DLC 安装方法和游戏安装方法一样】
