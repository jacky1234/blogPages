# 布局

  • flex: flex 类是一个Flexbox布局的类, 它使得该元素成为一个Flex容器
  • flex-col: 这个类设置了Flex容器的主轴方向为纵向(即垂直方向)
  • items-end: 这个类用于指定Flex容器的子元素在主轴方向(水平方向)上对齐到容器的底部
  • shrink-0: 这个类设置了元素的尺寸不会缩小。通常在Flexbox布局中, 元素会根据其内容和父容器的尺寸自动调整大小, 但是这个类可以防止元素缩小至其内容的大小
  • grow: 这个类将该元素的子元素在可用空间中平均分配, 使它们能够充满容器
  • items-center: 这个类用于指定Flex容器的子元素在主轴方向(水平方向)上居中对齐
  • justify-center: 这个类用于指定Flex容器的子元素在交叉轴方向(垂直方向)上居中对齐
  • min-h-screen: 这个类用于设置元素的最小高度为屏幕高度。在移动端和响应式设计中, 通常会使用这样的类来确保元素至少占据整个屏幕的高度
  • min-w-full: 这个类用于设置元素的最小宽度为 100%。这样, 表格将会水平填满其容器的宽度
  • md:h-screen: 这个类是响应式的类, 它表示在中等尺寸屏幕(md: medium)上, 元素的高度将设置为整个屏幕的高度
  • space-y-{x}: 这个类用于控制元素内部的子元素之间的垂直间距
  • py-{x}: 这个类用于设置元素的上下内边距(padding)
  • gap-{x}: 这个类设置了链接内部的元素之间的间隙为 {x} 个单位
  • self-start: 这个类用于指定链接自身在主轴方向(水平方向)上靠左对齐
  • pt-{x}: padding top
  • p-{x}: padding
  • md:p-{x}: 表示在中等大小屏幕以上内边距为 {x}
  • ml-{x}: margin left
  • rounded-full: 这个类将图片的边框设置为圆形, 使其呈现圆形的外观
  • divide-y: 这个类用于在元素的子元素之间垂直方向上添加分隔线
  • divide-{gray-300}/{x}: 这个类用于设置分隔线的颜色, x: 表示不透明度(0-100)
  • bg-red-500: 以 bg- 开头, 后跟颜色名称和对应的色调(如果有的话)。在这里, 表示红色背景, 色调为 500
  • hover:bg-blue-400: 这个类定义了鼠标悬停在链接上时背景颜色变化为浅蓝色的效果。
  • rounded-md: 这个类名将 <div> 元素的边框设置为圆角。md 表示在中等大小屏幕以上生效
  • hidden md:block: 这个类是用于控制元素的显示和隐藏。在这里, hidden 类将图片在小屏幕上隐藏, 而 md:block 类则在中等尺寸屏幕(md: medium)上显示图片。这样做可以根据屏幕尺寸来控制图片的可见性
  • md:w-2/5 md:px-20: 这个类是响应式的类, 它表示在中等尺寸屏幕(md: medium)上, 元素的宽度将会设置为父容器宽度的2/5, 而内边距(padding)为20个单位在水平方向

# 字体

text-color 内置可选颜色选项, 可以点击 链接 (opens new window) 查看

  • text-base: 这个类用于设置元素内文本的基本大小
  • text-sm: 这个类用于设置链接内部文本的字体大小为小号
  • text-white: 这个类用于设置链接内部文本的颜色为白色
  • text-xl: 设置了元素的文本大小为大型
  • leading-{x}: 这个类用于设置元素内文本的行高
  • font-medium: 这个类用于设置链接内部文本的字体粗细为中等
  • md:text-2xl: 则是在中等屏幕尺寸及以上时将文本大小增加到 2 倍大型

# other

  • sr-only: 被应用了 sr-only 类名的元素内容将在视觉上被隐藏, 但对于使用屏幕阅读器的用户来说, 这些内容仍然是可用的
