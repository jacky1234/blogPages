如何触发组件更新

11.如何触发组件更新

# 数据来源 （单向的）

  1. 来自父元素的属性
  2. 来自自身状态 data
  3. 来自状态管理 如 vuex Vue.observable

响应式更新原理： https://cn.vuejs.org/v2/guide/reactivity.html

# Demo

父控件传递info给子控件，子控件打印 info demo

//1
data() {
    return {
        info: {
            number: undefined   //提前声明，对 info下面字段进行响应式设置
        }
    }
}
this.info.number = 1; 

//2 
data() {
    return {
        info: {}
    }
}            
this.$set(this.info, "number", 1);
