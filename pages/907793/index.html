aapt

aapt（Android Asset Packaging Tool）是一个用于打包和管理 Android 应用程序资源的命令行工具。它可以执行多种操作，包括编译资源、打包 APK 文件、解析 APK 文件等。以下是一些常用的

# aapt 命令及其功能：

  • aapt dump badging <apk-file>：显示 APK 文件的清单信息，包括应用程序包名、版本信息、启动 Activity、权限等。
  • aapt list <apk-file>：列出 APK 文件中的所有文件和目录。
  • aapt crunch <input-dir> -C <output-dir>：压缩和优化指定目录中的资源文件。
  • aapt add <apk-file> <file1> [<file2> ...]：向 APK 文件中添加文件。
  • aapt remove <apk-file> <file1> [<file2> ...]：从 APK 文件中删除文件。
  • aapt package -f -M <AndroidManifest.xml> -I <input-file1> [<input-file2> ...] -F <output.apk> -S <resource-dir> -A <asset-dir>：根据提供的清单文件、资源目录和资产目录等生成 APK 文件。

这只是 aapt 工具的一些常见用法和命令示例。您可以通过运行 aapt 命令加上 --help 参数来获取更多详细的命令选项和用法说明。

注意：从 Android SDK 25 开始，aapt 已被 aapt2 替代为默认的资源编译和打包工具，但仍然可以使用 aapt 进行一些基本操作和与旧版本的 APK 兼容。
